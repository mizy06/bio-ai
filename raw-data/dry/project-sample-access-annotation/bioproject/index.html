NCBI 官方将其定义为与单一 initiative 相关的 biological data collection，用于汇总 diverse data types 的项目入口。

数据规模与更新情况

平台持续更新；本轮先按官方项目定位完成纳入，不误写未核总量。

数据格式与字段结构

核心对象可概括为 BioProject → linked dataset repositories / accessions。

数据内容特点与适用场景

适合作为 dry-lab 的 project → dataset 主线节点。

获取方式、开放性与可用性

官方项目入口公开可访问。
