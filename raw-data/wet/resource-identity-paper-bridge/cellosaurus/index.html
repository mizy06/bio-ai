Item 5.2

Cellosaurus

Resource Type

细胞系标准化数据库。

Unit

cell line accession

Primary Key

CVCL accession

完整内容

资源类型分类

细胞系标准化数据库。

展示层级

P0 细胞系身份层。

在湿实验资源图谱中的定位

用于细胞系命名标准化、别名对齐与污染风险提示。

推荐入库单元

cell line accession

推荐结构表

cell_line_record、cell_line_synonym

数据来源与覆盖领域

来自 Cellosaurus 官方站点，覆盖人、小鼠、大鼠及其他物种细胞系。

数据规模与更新情况

官方首页当前写明 Release 55 of March 2026，并给出 167,186 条 cell lines，其中 122,835 条 human、31,079 条 mouse、3,233 条 rat。

下载单元数量

若按细胞系 accession 口径统计，当前约 167,186 条 cell lines。

单个下载单元平均大小

按 5 个细胞系页面抽样，平均约 56.12 KB。

数据格式与字段结构

核心对象可概括为 cell line accession → recommended name / synonyms / species / flags / references；官方下载目录同时提供 txt、obo、xml、xsd 等格式。

数据内容特点与适用场景

适合 protocol 中细胞系实体规范化、细胞系别名对齐与问题细胞系识别。

获取方式、开放性与可用性

主页、说明页、单条细胞系页面和官方下载目录均可公开访问。

准入判断

主页、细胞系页面与下载目录抓取成功。

推荐批量主键

CVCL accession

官方批量入口

  • https://www.cellosaurus.org/
  • https://ftp.expasy.org/databases/cellosaurus/

单平台复核结论

是湿实验侧最稳定的细胞系身份层之一，数量、页面和下载目录均已核实。
