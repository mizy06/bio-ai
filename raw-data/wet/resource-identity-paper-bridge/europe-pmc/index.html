Item 5.4

Europe PMC

Resource Type

论文、注释与开放全文桥接平台。

Unit

PMCID / PMID / OA article

Primary Key

PMID / PMCID / Europe PMC id

完整内容

资源类型分类

论文、注释与开放全文桥接平台。

展示层级

P0 论文桥接层。

在湿实验资源图谱中的定位

用于 paper、methods、annotation 和开放全文之间的桥接。

推荐入库单元

PMCID / PMID / OA article

推荐结构表

literature_record、literature_annotation、literature_linkout

数据来源与覆盖领域

来自 Europe PMC Developers、REST 和 Downloads 体系，覆盖论文元数据、开放全文、注释和链接。

数据规模与更新情况

官方 REST 检索在 query=*、pageSize=1 口径下当前返回约 47,830,420 条 records；平台持续更新。

下载单元数量

若按 REST 检索记录口径统计，当前约 47,830,420 条。

单个下载单元平均大小

按 5 条 REST lite records 抽样，平均约 0.72 KB；若按开发页和下载页等说明页面抽样，页面级大小约 39.39 KB。

数据格式与字段结构

核心对象可概括为 article → metadata / annotations / full text / links。

数据内容特点与适用场景

适合把 protocol 与论文、全文、注释和外部数据链接回连起来。

获取方式、开放性与可用性

开发页、下载页和 REST 检索接口均可公开访问。

准入判断

开发页、下载页与 REST 检索抓取成功。

推荐批量主键

PMID / PMCID / Europe PMC id

官方批量入口

  • https://europepmc.org/developers
  • https://europepmc.org/RestfulWebService
  • https://europepmc.org/downloads

单平台复核结论

是湿实验侧最稳定的论文桥接入口之一，API 级批量抓取可行。

官方来源

https://europepmc.org/developers ，https://europepmc.org/RestfulWebService ，https://europepmc.org/downloads
