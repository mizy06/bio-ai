Resource Type

Unit

Primary Key

完整内容

资源类型分类

质粒、载体与 protocol 资源平台。

展示层级

P0 构建物与试剂层。

在湿实验资源图谱中的定位

用于把 protocol 步骤连接到真实构建物、载体和相关 protocol。

推荐入库单元

plasmid / protocol / collection

推荐结构表

construct_record、construct_protocol_link

数据来源与覆盖领域

来自 Addgene 官方平台，覆盖 plasmids、viral vectors、recombinant antibodies 以及 protocol 资源。

数据规模与更新情况

官方首页当前写明 170,838 个 plasmids、6,863 个 labs、1,100 个 ready-to-use viral vectors 和 258 个 recombinant antibodies。

下载单元数量

若按主资源单元 plasmid 统计，当前约 170,838 个。

单个下载单元平均大小

按 5 个公开 plasmid 页面抽样，平均约 221.15 KB。

数据格式与字段结构

核心对象可概括为 plasmid / collection → protocol / sequence / vector metadata。

数据内容特点与适用场景

适合把 protocol 中的构建物、载体和相关 protocol 页面连接到真实实验资源。

获取方式、开放性与可用性

主页、protocols 页、plasmid 页面可公开访问；更深层交互能力依平台自身功能开放程度而定。

准入判断

主页、protocols 页与代表性 plasmid 页面抓取成功。

推荐批量主键

plasmid id

公开入口与发现入口

  • https://www.addgene.org/
  • https://www.addgene.org/protocols/

单平台复核结论

适合作为构建物与 protocol 联动层，当前可稳定抓到公开资源页。
